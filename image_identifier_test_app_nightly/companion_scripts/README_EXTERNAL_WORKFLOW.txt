Image Identifier External Workflow Scripts

The app can generate scripts using your currently configured watch folder.
Open Advanced > External Workflow > Install / Refresh Companion Scripts.

Basic workflow:
1. Start Image Identifier.
2. Sign in.
3. Open Advanced > External Workflow.
4. Choose a watch folder.
5. Start Watching.
6. Export images from Photoshop, Lightroom Classic, Luminar Neo, or another editor into that folder.

Recommended action mode while testing: Ask Each Time.
